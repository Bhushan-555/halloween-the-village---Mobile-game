# Halloween The Village (Mobile Game)

A mobile horror-themed game developed using Unreal Engine 5.

##  Features
- AI-driven enemy system
- Third-person character movement
- Dynamic environment and lighting
- Optimized for mobile gameplay

##  Tech Used
- Unreal Engine 5 (Blueprints + C++)
- Mobile optimization techniques
